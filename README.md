
# **Air Quality Control System

## Atmel SAM3X8E ARM Microcontroller Based all info along with demo video is in readme.md

This project is based on  air quality measurement which has become such an important thing now days during the pandemic. As Covid is airborne it is good to know whats in the air around us. The main purpose is to determine air quality and temperature in surrounding area.
